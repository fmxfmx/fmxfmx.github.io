这个示例就是平时我们头文件的样子。如果我们需要自定义这个内容，我们需要创建一个名字为 IDETemplateMacros 的 plist 文件，文档中提供了多个该文件的存放路径，我选择了 ~/Library/Developer/Xcode/UserData/ 这个路径，这样创建的所有文件都会是我自定义的样子.

我们可用的字段也是有限的，同样也能在 Xcode help 中找到，具体位置在 Xcode help -> Reference -> Text Macros 中，里面注释的也很清楚。
